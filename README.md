# campusNavigatorIA

| **Sección del README**        | **Contenido**                                                     |
|-------------------------------|-------------------------------------------------------------------|
| **Título del trabajo**        | PA4 – CampusNavigator IA: Recursividad y Grafos.                  |
| **Integrantes**               | Velásquez Fuero Nicoly Dayana, Salas Tucto Nadia Salisa, Bedon Saboya Carlos Josué |
| **Propósito**                 | Este repositorio contiene la solución del caso integrador **CampusNavigator IA**, un módulo que orienta a los estudiantes dentro del campus universitario. El sistema resuelve tres necesidades:**Ordenar** recorridos y resultados mediante una estrategia recursiva,**Modelar** las rutas entre ambientes, laboratorios y servicios usando **grafos**,**Recomendar** recorridos según el objetivo del usuario aplicando **DFS, BFS y A\***. La solución integra los temas de las sesiones 11 a 13: recursividad, principio divide y vencerás, ordenamiento recursivo (Merge Sort), teoría de grafos y algoritmos de búsqueda.|
| **Instrucciones de ejecución**| Cómo abrir y ejecutar cada clase Java en NetBeans.               |
| **Conclusiones del grupo**    | La recursividad y el principio divide y vencerás permiten descomponer problemas grandes en subproblemas idénticos más pequeños, obteniendo soluciones eficientes como Merge Sort (O(n log n)). La teoría de grafos ofrece el lenguaje para modelar la realidad del campus, y los recorridos DFS, BFS y A\* permiten explorarlo y recomendar rutas según el objetivo del usuario. Elegir la estructura y el algoritmo correctos determina la eficiencia, la memoria y la calidad de la solución. |
